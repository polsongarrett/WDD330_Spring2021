Chapter 9

The window.location property contains the following information about the current window:

  • href
  • protocol
  • host
  • port
  • pathname
  • search
  • hash
  • origin

The window.history property has useful information and methods to access the user's browsing history.

The window.screen property stores information about the browser window.

The window document has the cookie property, which is useful for peronalizing user experience. The cookie property has several useful methods and properties.

The window object also provides timeout, interval, and animation methods, but use CSS for animation effects when possible.

Chapter 11

The data- attribute can be assigned to an element and hold custom information. It is only meant to be used with JavaScript. 


  <div id='hero' data-powers='flight superSpeed'>
    Superman
  </div>  

  const superman = document.getElementById('hero');
  const powers = superman.dataset.powers;
  << 'flight superSpeed'

HTML5 currently has some APIs that make some tasks easier for developers:

  • Web Storage: Allows the browser to store information in localstorage and session storage.
  • Geolocation: Can obtain the grographical location of a device.
  • Web Workers: Allows multiple processes to work in the background.
  • Service Workers: Lets a worker script to run in the background while being able to intercept network requests. This allows users to still interact with the application if the network is offline.
  • Websockets: HTTP protocol only sends responses to requests, so if you had a chat app that worked this way, you could only responses if you texted first. Websockets allow constant two-way communication so that a connection is kept open.
  • Notifications: Lets the application display messages via the operating system.
  • Multimedia: HTML5 introduced the <audio> and <video> tags for inserting media on a page, as well as a Media API for playing back videos using JavaScript.
