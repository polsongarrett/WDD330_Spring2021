Chapter 8

CSS3 possesses the transform property, which allows us to use transform functions to rotate, translate, scale, and skew elements. This property also lets us change the transform-origin of an element.

Transitions are useful for gradually changing an element over time. This property has control over the start and finish of a transition, but not the intermediate states.

Animations have an advantage over transitions in that they can control the intermediate transitions using keyframes.

Chapter 12

The Canvas API is meant for drawing images with JavaScript by manipulating pixels. It can improve performance by decreasing the amount of downloaded resources such as images and videos. Images can be drawn in 2D and 3D by combining WebGL with HTML5 Canvas. We can also add image files, video files, and text to canvas elements.

Standard Vector Graphics (SVG) is a file format for describing vector graphics using XML. Inkscape is useful for creating SVGs in order to avoid having to tediously create an image from code. A JavaScript library called Raphaël makes drawing and animating with SVG more convenient.

Canvas operates in immediate mode, meaning that once a shape is drawn, it can't be modified again because it does not persist as an object. It's also unavailable in the DOM. However, it can be saved as a PNG or JPEG, and it is faster than SVG.

SVG operates in retained mode, meaning it is preserved because of the XML file in which it has been defined. SVG is accessible in the DOM, and it has tools like Raphaël and Inkscape. However, SVGs can't be manipulated the same way as pixels on canvas.

The Drag and Drop API lets us define elements as draggable and how they interact with other elements when dragged or dropped over them.
