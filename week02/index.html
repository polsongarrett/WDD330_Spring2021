Chapter 2

If you assign a non-primitive value to a variable, then any changes to that variable are will affect all references to that variable's value: 


  const c = { value: 1 }; 
  let d = c; // c.value = 1, d.value = 1 
  d.value = 2; // c.value = 2, d.value = 2

JavaScript creates wrapper objects for primitive data types, which makes it seem like primitive variable types have properties and methods.

This means we can treat primitives like they are objects using dot notation. For example, every string has a length property that tells us how many characters are in the string: 


  const name = 'Alexa'; // declare and
                        // assign a variable 
  << 'Barry' 
  name.length; // retrieve the name  
              // variable's length property 
  << 5

Chapter 3

Destructuring arrays allows us to assign multiple values at the same time, using arrays: 


  const [x,y] = [1,2];

Maps vs. Objects

  • Objects are limited to using strings for key values, whereas maps can use any data type as a key.
  • There is no efficient way to find the number of key-value pairs an object has, whereas this is easy to do with maps using the size property.
  • Objects have methods that can be called and prototypes that can be used to create a chain of inheritance whereas maps are solely focused on the storage and retrieval of key-value pairs.
  • The value of an object's properties can be accessed directly, whereas maps restrict you to using the get() method to retrieve any values.

Chapter 4

Arguments are passed when calling a function, parameters are set when defining a function.

The rest operator (...args) can be used to deal with multiple arguments by creating an array of arguments that are available inside the body of the function. It will collect all the arguments entered into an array. 


  function rest(...args){ 
    return args; 
      }

map() Iterates through an array, passes each element through a callback, replaces the element with the returned value from the callback

reduce() Iterates through an array and returns the sum of all the values in the array.

filter() Iterates through an array and returns a new array that only contains the elements that return true when passed to the callback.
